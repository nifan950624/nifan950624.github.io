使用插件

vscode重点就是数不尽的插件,下面介绍下自己安装的一些插件:

Chinese

中文，没什么说的，下载后第一个安装的

Vetur

使用 Vue，必定会安装Vetur,可以识别 vue 文件，带格式化功能(但最好不用这个格式化)

Auto Close Tag

帮你自动闭合标签，有时不是很准确，不过还是可以的。

Auto Rename Tag

帮你自动更新标签，在标签结尾使用没用，在开始标签使用。

Color Picker

颜色选择器，没什么用，可以出现颜色样板，滑动去色

Css Peek

点击 html 会自动跳转到对应的 css 位置

Dracula Official

最喜欢的一款皮肤，连顶部的 Tabs 栏都都标记颜色，真的很棒。

Manta’s Stylus Suppremacy

stylus 的格式化工具，自定义强。

打开用户设置，点开右上角的文本配置，写下代码:

Prettier-code formatter

格式化软件，支持语言超级多，而且自定义强，定制自己的代码风格，
Vuter 加它，完美解决各种不喜欢，个人不喜欢冒号和引号，所以这个
插件，让我找回新的感觉。

Markdown Preview

markdown 编写插件，可以让你的 vscode 瞬间变成记笔记写博客的软件，
无需再别的。

最后附上个人的 vscode 配置

可以直接粘贴到自己的配置，然后安装，暗色的软件，再根据自己的需求来调整
