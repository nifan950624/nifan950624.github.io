React的声明周期

Initialization >>> 初始化

Mounting >>> 挂载

  • componentWillMount >>> 在组件即将被挂载到页面之前执行

  • componentDidMount >>> 页面第一次被加载完了

Updation >>> 更新(props或者states变化)

共同：

  • shouldComponentUpdate >>> 组件更新之前会被执行，返回Boolean

  • componentWillUpdate >>> 组件被更新之前，他会执行 如果shouldComponentUpdate为true才会执行

  • render执行

  • componentDidUpdate >>> 渲染完毕后，被执行

不同点:

  • componentWillReceiveProps(1. 当一个组件从父组件接受参数 2. 如果这个组件第一次存在于父组件中，不会执行 3. 如果这个组件已经存在于父组件，就会执行)

`componentWillUnMounting`` >>> 当组件即将被移出的时候执行
